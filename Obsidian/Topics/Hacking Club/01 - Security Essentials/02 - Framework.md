Existem frameworks que auxiliam na criação do time de segurança

NIST CSF (Cyber Security Framework)
O CSF fornece uma documentação para auxiliar no mapeamento da sua empresa, oferece toda uma metodologia para com os passos necessario para enumerar toda a empresa.

OWASP SAMM
Segurança de aplicações visa medir o maturidade no ciclo de desenvolvimento de aplicações.

CIS Control
Oferece quais controles devem ser implementados nos nosso servidores, rede, autenticação navegadores. Ele oferece uma ferramenta para ser executada no nosso ambiente e retorna na oque precisa ser feito
