A Linux Foundation é a organização que reúne muitos projetos críticos de código aberto que impulsionam corporações, abrangendo todos os setores da indústria. Seu trabalho hoje se estende muito além do Linux, promovendo a inovação em todas as camadas do conjunto de software.
### Linux Kernel
**The Red Hat Family** tem as  distros CentOS, Fedora e Oracle Linux como distribuições

**The SUSE Family** Linux Enterprise Server, ou SLES é semelhante á relação entre o Red HAT.

**The Debian Family** o debian é a distribuição mais conhecida, serve para outras distribuições como Ubuntu, linux Mint entre outros.
